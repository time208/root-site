# [Demo](https://lch&#122;h&#51;473.github.io/nickitem/index "QQ群聊代码生成器")

## 简介

如题，生成QQ彩色群昵称绝版代码的小工具(仅移动端QQ本体可见)

食用步骤：

1. 按照网页内提示生成个性化代码；
2. 将代码复制到QQ群昵称修改页面的输入框内；
3. 输入自己的昵称然后保存，回到聊天页面；
4. 出现指定的个性化效果，设置成功
